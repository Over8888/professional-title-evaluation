# 职称评审系统前端

本目录为职称评审系统的前端工程，提供活动配置、候选人管理、评委管理、规则配置、投票入口和结果统计等管理界面。

## 技术栈

- Vue 3
- Vite
- Element Plus
- Pinia

## 常用命令

```bash
npm install
npm run dev
npm run build:prod
```

## 目录说明

- `src/views/evaluation`：职称评审业务页面
- `src/api/evaluation`：职称评审业务接口
- `src/layout`：后台管理布局
- `src/components`：通用组件
